О проекте

Индивидуальная вариативная часть учебной практики

Описание

Данный мини-проект был реализован как вариативная часть учебной проектной практики. Он представляет собой образовательный сайт с простыми играми и интерактивной проверкой знаний по IT-терминам на английском языке.

Цель

Цель проекта — создать лёгкую обучающую платформу, которая сочетает справочную информацию и простую геймификацию, позволяющую лучше запомнить термины из мира информационных технологий.

Что реализовано

  • Самописный HTTP-сервер на Python
  • HTML-интерфейсы: главная страница, словарь, викторина
  • JS-логика викторины с динамической генерацией вопросов
  • CSS-оформление в едином стиле
  • Адаптация под локальный запуск (через `server.py`)

Как использовать

Для запуска сайта достаточно выполнить команду python server.py в каталоге проекта, после чего сайт будет доступен по адресу http://localhost:8080.
